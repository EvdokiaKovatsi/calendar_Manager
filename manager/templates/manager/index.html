    Mass import and export tool for Google Calendar

    You need to login with a Google account in order to use the import and export features.
    You can view the help page without login in to find out more about the process.

    Import csv

    You can use this page to import multiple events to one of your Google Calendars. Pick the calendar you want to add the events to and upload a .csv file with the correct format. There are more detailed instructions under the form and you can visit the help page to check the format for the .csv file.

    Export csv

    You can use this page to download a .csv file with any events between two dates. Pick the calendar and the desired dates and you can download a .csv file with any events between them. The format of the .csv file is created automaticaly.

    Help page

    This page has information about the format of the .csv file you can upload.
